== Description ==

Adds a quick tool to manually flush the rewrite rules.

`Tools` -> `Flush Rewrite Rules`

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the folder `kbj-flush-rewrite-rules` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. The `Flush Rewrite Rules` tool is listed under the Tools menu in the admin.
2. Rewrite Rules flushed successfully.
